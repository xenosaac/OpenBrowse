Design Details

Header
Emerald ✦ mark replaces purple. Status pips glow with emerald box-shadow. Icon buttons get emerald glow on hover.
Messages
User bubbles: emerald-tinted glass (not purple gradient). Agent bubbles: subtle white glass. Action messages: left emerald border accent. No avatars — tiny colored dots instead.
Composer
Glass input field that gains emerald border on focus. Solid emerald send button with glow on hover and press depth. Arrow ↑ icon instead of text.
Run Context
Emerald-tinted glass card with pulsing status dot. Subtle, non-intrusive. Shows current agent action.
Divider Line
Sidebar/main content divider: 1px rgba(255,255,255,0.05). Draggable with col-resize cursor. No visible handle — pure line.
